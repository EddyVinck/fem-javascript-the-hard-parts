Principles of JavaScript

There is a global execution context. It is created when we run our code.
Thread of execution: parsing and executing the code line after line
Live memory of variables with data (global variable environment).

When you run a function you create a new (local) execution context and a local memory (variable environment) where anything defined in the function is stored.

Call Stack. When a function is invoked it is pushed to the call stack. When the function returns or finds the closing curly brace (implicit return) the function is popped off the call stack.
